Aplikacja pogodowa stworzona w React jako projekt zaliczeniowy na studiach. Wyświetla aktualną pogodę i prognozę 5-dniową dla dowolnego miasta na świecie.

Projekt został stworzony w ramach zaliczenia przedmiotu **Programowanie Frontend** na studiach. Główne cele:

- Praktyczna implementacja biblioteki **React** z hookami
- Integracja z zewnętrznym **API** (OpenWeatherMap)
- Zarządzanie stanem aplikacji za pomocą **Redux**
- Implementacja routing z **React Router**
- Zapisywanie danych w **localStorage**


| Technologia | Wersja | Cel użycia |
|------------|--------|------------|
| **React** | 18.2.0 | Główna biblioteka frontendowa |
| **Redux Toolkit** | 1.9.7 | Zarządzanie globalnym stanem |
| **React Router** | 6.20.0 | Nawigacja między stronami |
| **Axios** | 1.6.2 | Komunikacja z API |
| **OpenWeatherMap API** | - | Dane pogodowe |
| **CSS3** | - | Stylowanie komponentów |
